# Dayu Xia - Personal Website

This repository hosts my personal website built with the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) Jekyll theme.

## Main Sections

- Home: pinned introduction and latest posts
- About: personal background and research interests
- CV: education, experience, awards, and projects
- Portfolio: selected travel and photography posts

## Local Development

```bash
bundle install
bundle exec jekyll s
```

The site will be available at `dyxia1241.github.io`.
